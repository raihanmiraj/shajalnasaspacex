 # AloronBD

It is Complete Quiz Project. 

## Installation

Just clone this , and run 

```bash
npm start
```


## Features

- Quiz Timer
- Question/Options Soring 
- Once click answer can't remove
- Student Dashboard, can check result and answer
- Student profile Settings
